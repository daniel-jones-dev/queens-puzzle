# Queens puzzle solver

Solver for the Queens puzzle, found on LinkedIn.

## Puzzle archive

https://www.playqueensgame.com/ has an archive of all puzzles that appeared on LinkedIn.

You can retrieve puzzles from https://www.playqueensgame.com/api/daily?date=YYYY-MM-DD.

https://queensstorage.blob.core.windows.net/puzzles/linkedinPuzzles.json is another option, but incomplete.
