Solver rules

Every deduction the solver can make, easiest first. If no rule applies, the puzzle requires guessing.

Easy Forced single-cell placements
Single cell in row
Every row must contain exactly one queen. If all but one cell in a row has been eliminated, the remaining cell must be the queen for that row.
Example
Row 3 has one cell left — place the queen.
Single cell in column
Every column must contain exactly one queen. If only one cell in a column remains available, it must be the queen for that column.
Example
Column 1 has one cell left — place the queen.
Single cell in region
Every colour region must contain exactly one queen. If a region has only one uneliminated candidate left, that cell must be the queen.
Example
The orange region has one cell left — place the queen.
Medium Region confinement propagation
Region spans one row
If all remaining candidates for a colour region lie in the same row, the region's queen must be in that row. All other regions' cells in that row are eliminated.
Example
Blue's candidates are all in row 2 — eliminate non-blue cells in that row.
Region spans one column
If all remaining candidates for a colour region are in one column, all other cells in that column from other regions are eliminated.
Example
Green's candidates are all in col 3 — eliminate non-green cells in that column.
Hard Multi-region exclusion
Exclusion chain
When N colour regions together have all their candidates in the same N rows (or columns), those rows are fully claimed by those regions. All other regions' cells in those rows are eliminated.

The two-region case is the most common: two regions whose candidates only appear in two rows together force both queens into those rows.
Example
Green + orange only span rows 0–1 together. Blue cells in those rows are eliminated.
