Solver rules

Every deduction the solver can make, grouped by difficulty. Easy rules are applied first; the puzzle is considered to require guessing if no rule applies.

Easy Forced placements from a single row, column, or region
Single cell in row
Every row must contain exactly one queen. If all but one cell in a row has been eliminated (marked ✕), the remaining cell must be the queen for that row.
Example
Row 3 has one cell left — place the queen.
Single cell in column
Every column must contain exactly one queen. If only one cell in a column remains available, it must be the queen for that column.
Example
Column 1 has one cell left — place the queen.
Single cell in region
Every colour region must contain exactly one queen. If a region has only one uneliminated candidate remaining, that cell must be the queen for the region.
Example
The orange region has one cell left — place the queen.
Medium Region confinement propagation
Region spans one row
If all remaining candidates for a colour region lie within the same row, then the region's queen must be in that row. No other region can place its queen there, so all other non-region cells in that row can be eliminated.
Example
Blue's candidates all sit in row 2 — eliminate all non-blue cells in row 2.
Region spans one column
The same logic applied to columns: if all remaining candidates for a colour region are in one column, then all other cells in that column from other regions can be eliminated.
Example
Green's candidates all sit in column 3 — eliminate all non-green cells in col 3.
Hard Multi-region exclusion
Exclusion chain
A generalisation of region confinement. When N colour regions together have all their remaining candidates confined to the same N rows (or columns), those rows are fully "claimed" by those regions — no other region can place a queen there. All other regions' cells in those rows are eliminated.

The two-region case is the most common: two regions whose candidates between them only appear in two rows force both queens into those rows, leaving the rest of those rows clear for nothing else.
Example
Green + orange together only span rows 0–1. Blue cells in those rows are eliminated.
